add commits in my local repository
long time ago, I made a commit in my local repository. I want to add more commits to it. How can I do that?
To add more commits to your local repository, you can follow these steps:
1. Make changes to your files in your local repository using your preferred text editor or IDE.
2. Use the command `git add <file_name>` to stage the changes for commit.
3. Use the command `git commit -m "Your commit message"` to commit the changes with a descriptive message.
4. Repeat steps 1-3 for each set of changes you want to commit.
5. Once you have made all your commits, you can use the command `git log` to view the commit history and verify that your new commits have been added to the local repository.
